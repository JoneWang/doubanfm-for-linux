=====================
Douban FM
=====================

# System environments
--

* System: ubuntu 12.04 LTS (used notify-osd)

* Software: OpenJDK Java 6 Runtime (decoder)


# Began to enjoy music
--

    $ chmod +x doubanfm

    # run
    $ ./doubanfm

    # or ./doubanfm [cmd] [channel_id]
    $ ./doubanfm c 1001400

# Command
--

c : select channel id
